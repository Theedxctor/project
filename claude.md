Project Prompt: StrathFood Full-Stack Implementation1. Project OverviewAct as a Senior Full-Stack Developer. Your goal is to build StrathFood, a web application designed to eliminate long queues at the Strathmore University cafeteria. The system allows students and staff to browse time-sensitive menus, place orders, pay via M-Pesa, and receive a unique code for food pickup.Core MissionReduce waiting times which currently cause customers to abandon purchases after 5 minutes.Enhance the student experience by allowing pre-orders.Provide a secure, authenticated platform restricted to the Strathmore community.2. Technical StackBackend: Laravel 11 (Headless API) with Sanctum for authentication.Frontend: Next.js (App Router) with Tailwind CSS.Database: MySQL.Payments: M-Pesa Daraja API.3. Database Schema & ModelsGenerate migrations and Eloquent models for the following entities:Users: Include name, email (must end in @strathmore.edu), password, and role (student/staff/admin).Foods: Include name, price, image_path, category (Morning/Evening), and is_available.Orders: Include user_id, total_price, status (Pending, Paid, Ready, Picked Up), and a unique 6-character pickup_code.Order_Items: Pivot table for many-to-many relationship between orders and foods.4. Backend Requirements (Laravel API)Implement the following logic:Restricted Registration: Validation logic to ensure only Strathmore emails are accepted.Time-Sensitive Menu API: An endpoint that returns "Morning" meals if the current time is before 11:00 AM and "Evening" meals thereafter.Order Flow:Create a "Pending" order when a user checkouts.Generate a unique pickup_code only after successful payment.Admin Dashboard API: Endpoints for cafeteria staff to view active orders, update status to "Ready," and delete users or menu items.Notifications: Logic to trigger alerts every 15 minutes for one hour once an order is ready.5. Frontend Requirements (Next.js)Build a responsive UI based on the project wireframes:Authentication Pages: Secure Login and Registration forms using your Strathmore credentials.Interactive Menu: * Display food items with images, prices, and portion sizes.A "Cart" component to manage selections before final submission.Student Dashboard: * Display active order status.Show the generated pickup code clearly once the order is ready.Admin Interface: A real-time table for staff to manage orders and inventory.
